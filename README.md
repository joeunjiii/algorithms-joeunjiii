# algorithms-joeunjiii

## 목표 : 꾸준함
이 레포는 “문제를 많이 풀자”가 아니라, **하루에 2~3문제씩 꾸준히** 푸는 습관을 만들기 위해 만들었습니다.  
저는 끈기가 약한 편이라서, 욕심을 크게 내기보다 **작게라도 매일 해내는 것**을 목표로 잡았습니다.

---

## 규칙
- 하루 **2~3문제**를 푸는 걸 기본으로 합니다.  
  (2문제는 기본, 1문제는 보너스 느낌으로)
- 시간이 없는 날은 **쉬운 1문제라도** 풀고 끝냅니다. “0”을 만들지 않는 게 더 중요합니다.
- 하루 놓쳤다고 흐름을 끊지 않습니다. 다음 날 **그냥 다시 시작**합니다.
- 컨디션이 좋은 날이든 나쁜 날이든, 꾸준히 하다 보면 결국 실력이 따라온다고 믿습니다.

---

## 언어 운영 전략
꾸준함이 목표라서, 언어도 복잡하게 운영하지 않으려고 합니다.

- 기본은 **주력 언어 1개**로 풉니다. (매일 2~3문제는 여기서)
- 루틴이 안정되면, 여유 있을 때만 **같은 문제를 다른 언어로 한 번 더** 풀어봅니다.  
  같은 문제라도 언어가 바뀌면 구현 방식이 달라져서 도움이 됩니다.
- 다만 루틴이 흔들릴 것 같으면, 보조 언어는 과감히 내려놓고 **주력 언어만** 갑니다.

---

## 나에게 하는 약속
- 완벽하게 하려고 멈추지 않기.  
- 오늘 2문제만 풀어도 충분하다고 인정하기.  
- 중요한 건 “많이”가 아니라 **내일도 계속하는 것**이라는 걸 잊지 않기.

작게라도 매일 쌓이면, 언젠가는 분명히 큰 차이가 난다는 것
